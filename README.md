## Installation 

install requirements: 
`./install_reqs.sh`

setup with:  
`./setup.sh`

source the local `bashrc.sh` in your actual `.bashrc` by adding this line: 
`source /home/elijah/.config/config/bashrc.sh`

## configure i3/config
* configure displays
```
xrandr
```

* configure trackpad input device
```
xinput list
```
